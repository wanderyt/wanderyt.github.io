offset vs client vs scroll

发表于 2018-02-05 | 分类于 UI Development

先上一张神图：

以height为例：

offsetHeight

MDN Definition

offsetHeight = 可看到的元素高度 + 纵向padding + 边框高度 + 横向滚动条高度

clientHeight

MDN Definition

clientHeight = 可看到的元素高度 + 纵向padding - 横向滚动条高度

scrollHeight

MDN Definition

scrollHeight = 可看到的元素高度 + 看不到的元素高度 + 纵向padding
